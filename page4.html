HTML File path:

In HTML, file paths tell the browser where to find resources like images, CSS, JS, or other HTML files.

Types of File Paths:

a)Absolute Path
  1. An absolute path is the full URL of a resource.
  2. Browser ignores your file location; it goes straight to the web address.
<img src=https://www.example.com/images/logo.png" alt="Website Logo">
b)Relative Path
  1. Relative path depends on the location of your HTML file.
  2. Makes your site portable (works on any server/folder structure).

Types of relative paths:

  • Same Folder
    <img src="logo.png" alt="Logo">
  • Subfolder
    <img src="images/logo.png" alt="Logo">
  • Parent folder
    <img src="../logo.png" alt="Logo" >
  • Nested folder
    <img src="../assets/images/logo.png" alt="Logo" >

Class

  • A class is used to group multiple elements that share the same styling.
  • Defined using class="name" in HTML.
  • In CSS, use . to target a class.

ID

  • An ID is unique; only one element should have a particular ID.
  • Defined using id="unique-name" in HTML.
  • In CSS, use # to target an ID.
Comparison between Class and ID:-
Feature Class ID
Uniqueness Can be used on multiple elements Must be unique; only one element
HTML Syntax class="name" id="unique-name"
CSS Selector .classname { } #idname { }
Purpose Group multiple elements with same style/behavior Target a single specific element
