HTML:

HTML (HyperText Markup Language) is the standard language used to create web pages. It tells the browser what to display on the screen (text, images, links, videos, forms, etc.). In simple words: HTML is the backbone/structure of every website.

HyperText:

 This refers to the ability to create links that connect web pages to one another, making the web a connected "web" of information.

Markup Language:

 This means you use "tags" to surround your content, giving that content meaning and structure. You are "marking up" a plain text document.

Tags in HTML:

In HTML, a tag is a piece of code written inside angle brackets (< >) that tells the browser how to structure and display content. Tags are the building blocks of HTML.

Some HTML tags:

1.The Heading (<h1> to <h6>)

Headings are tags used to define titles and subtitles on your page. They are crucial for creating a logical hierarchy and outline for your content.

Syntax:

<h1>Heading 1</h1>

<h2>Heading 2</h2>

<h3>Heading 3</h3>

<h4>Heading 4</h4>

<h5>Heading 5</h5>

<h6>Heading 6</h6>

2.The Anchor Tag (<a>)

 The Anchor tag is what makes the web "hypertext." It is used to create a hyperlink to another webpage, a file, or a location within the same page.

Syntax: <a href="https://www.coderarmy.in/">Vist Coder Army</a>

3.Paragraph Tag <p>

The paragraph tag is the most common tag you'll use. It's for grouping sentences and blocks of text together.

Syntax: <p>This is the first paragraph. It contains several sentences about a topic. <p/>
<p> This is a second, separate paragraph. The browser will display it with a space between it and the first one.<p/>

4. Line Break <br> Tag

The <br> tag is used to insert a line break.
Syntax:
Hello <br> World

5. Horizontal Rule <hr> Tag

The <hr> tag is used to insert a horizontal line that separates content.
