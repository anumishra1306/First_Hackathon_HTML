Internet:

The Internet is a global network of interconnected computers that communicate using a common set of protocols (mainly TCP/IP).

  1. It connects billions of devices worldwide (computers, smartphones, servers, IoT devices).
  2. Uses physical infrastructure (cables, satellites, Wi-Fi) and logical infrastructure (protocols, IP addressing).

How is Data Transferred?

  • Information (e.g., a webpage) is broken into small packets.
  • Each packet contains: source address, destination address, and data.
  • Packets travel through routers/switches, possibly taking different paths.
  • At the destination, packets are reassembled into the original message.

Protocols:

  • TCP (Transmission Control Protocol): Ensures reliable delivery, correct order, error checking.
  • UDP (User Datagram Protocol): Faster, but no reliability (used in video streaming, gaming).

What is an IP Address?

An IP address (Internet Protocol Address) is a unique identifier for a device on a network.

  • Example: 192.168.1.1 (IPv4), 2001:db8:85a3::8a2e:370:7334 (IPv6).
  • Without IPs, computers wouldn’t know where to send/receive data.

Like a home address for your computer on the internet.

How to Access a Website

Steps when you type www.example.com into a browser:

  • Browser checks its cache/DNS for the IP of example.com.
  • If not found, DNS resolves it (explained later).
  • Browser sends an HTTP/HTTPS request to the server IP.
  • Server processes the request and sends back HTML, CSS, JS.
  • Browser renders the page for you.

MAC Address:

A MAC (Media Access Control) Addressis a unique, permanent serial number burned into every network-capable piece of hardware (your laptop's Wi-Fi card, your phone, your smart TV, the network port on your desktop). To track you, if you get offline and someone get the ip, they will get your message. A MAC address is a 48-bit number. To make it readable for humans, it's typically written as 12 hexadecimal digits.

  • Hyphen-Separated (Common on Windows): 3C-22-FB-A3-B4-C5
  • Period-Separated (Used by Cisco and other network gear):3c22.fba3.b4c5
  • No Separators (Less common, seen in some software):3C22FBA3B4C5
  • Colon-Separated (Most Common):3C:22:FB:A3:B4:C5

Port Number:

Port numbers are numerical identifiers (0-65535) in computer networking that identify specific applications or services on a device. They work with IP addresses to direct network traffic to the correct service on a machine, with common ports assigned to standard applications like HTTP (port 80) and HTTPS (port 443).

Example:
  • HTTP → Port 80
  • HTTPS → Port 443
  • FTP → Port 21
Range: 0-65535
  • 0–1023: Well-known ports
  • 1024–49151: Registered ports
  • 49152–65535: Dynamic/private ports

DNS : Domain Name System

DNS is the global, distributed system that translates the human-friendly domain names into the computer-friendly IP addresses. Without it, you'd have to memorize hundreds of IP addresses to browse the web.

  • Converts domain names (google.com) into IP addresses (142.250.64.78).
  • So humans don’t need to memorize numbers.

In-depth Working of DNS

Example: Visiting www.facebook.com

  • Browser → asks local DNS resolver.
  • Resolver → checks cache; if not found, queries root server.
  • Root server → tells which TLD server to ask (.com).
  • TLD server → points to Facebook’s authoritative server.
  • Authoritative server → returns IP 157.240.229.35.
  • Browser →contacts that IP → loads website.

What is IPv4?

IPv4 (Internet Protocol Version 4) is the fourth version of the Internet Protocol, introduced in 1983.

  • It uses a 32-bit address → about 4.3 billion unique addresses.
  • Example: 192.168.1.1

IPv4 Features:

  • Widely used, but address shortage is a major issue.
  • Supports manual and DHCP (dynamic) configuration.
  • Provides simple addressing and routing.

What is IPv6?

IPv6 (Internet Protocol Version 6) is the latest version, developed to solve IPv4 exhaustion.

  • Uses 128-bit address → about 340 undecillion addresses (virtually infinite).
  • Example: 2001:0db8:85a3:0000:0000:8a2e:0370:7334

IPv6 Features:

  • Larger address space (solves IPv4 shortage).
  • Built-in security (IPSec).
  • Better support for mobile devices and IoT.
  • More efficient packet processing (no NAT required).

Local Area Network (LAN)

A Local Area Network (LAN) is a network covering a small geographical area, such as a home, office, or school. It connects devices like computers, printers, and servers, allowing high-speed communication and resource sharing.

Switch & Router

  • Switch: Connects devices within a LAN and forwards data based on MAC addresses.
  • Router: Connects different networks (LAN to the internet) and forwards data using IP addresses.

Public and Private Addresses

  • Private IP: Used within LANs (e.g., 192.168.x.x, 10.x.x.x) and not routable on the internet.
  • Public IP: Globally unique, provided by ISPs, and used for internet communication.

Routers use NAT (Network Address Translation) to map private IPs to public IPs.

Modern Routers

Modern routers combine the functions of a switch, router, firewall, and wireless access point. They provide DHCP services, built-in security (firewalls, encryption), and support for both IPv4 and IPv6.

Web Devlopment:

Definition

Web development is the process of creating and maintaining websites and web applications that run on the internet.

Types of Web Development

  • Front-End (Client-Side): Focuses on what users see in their browser. Uses HTML, CSS, and JavaScript.
  • Back-End (Server-Side): Handles logic, databases, and server communication. Uses PHP, Node.js, Python, Java, etc.
  • Full-Stack Development: Combines both front-end and back-end development.

What is the MERN Stack?

The MERN Stack is a popular set of technologies used for full-stack web development. It is called “MERN” because it includes MongoDB, Express.js, React, and Node.js.
