6. List Elements (<ul>, <ol>, <li>)

These tags are used to create lists. This is a perfect example of nesting.

The Tags:

  • <ul>: An Unordered List. It creates a bulleted list.
  • <ol>: An Ordered List. It creates a numbered list.
  • <li>: A List Item. Each item in either type of list must be wrapped in an <li> tag.

Example:

Unordered List:

<ul>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>

Ordered List

<ol>
  <li>Step 1: Open your browser</li>
  <li>Step 2: Type website URL</li>
  <li>Step 3: Press Enter</li>
</ol>

7.Nested List:

A nested list means a list inside another list. You can nest unordered lists (<ul>) or ordered lists (<ol>), or even mix both.

Example 1: Nested Unordered List
  • Fruits
    • Apple
    • Mango
    • Banana
  • Vegetables
    • Carrot
    • Potato
    • Tomato
Example 2: Nested Ordered List
  1. Frontend
    1. HTML
    2. CSS
    3. JavaScript
  2. Backend
    1. Node.js
    2. Express.js

8.HTML table:

Tables in HTML are used to display data in rows and columns. They are built using tags like <table>, <tr>, <th>, and <td>.

Header 1 Header 2 Header 3
Row 1, Col 1 Row 1, Col 2 Row 1, Col 3
Row 2, Col 1 Row 2, Col 2 Row 2, Col 3
  • <table> starts the table.
  • The first <tr> contains <th> elements → table headers.
  • The next <tr> elements contain <td> → table data.
